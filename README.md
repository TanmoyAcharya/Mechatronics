An interactive electrical machines simulation platform for learning and understanding electrical machines through real-time simulations, animations, and deep physics explanations.

## 🎯 Features

### Machine Simulators

| Machine | Description |
|---------|-------------|
| **Synchronous Machine** | Motor/Generator with field excitation, phasor diagrams, power factor control |
| **Induction Machine** | Asynchronous motor/generator with torque-speed curves, slip analysis |
| **DC Motor** | Shunt, Series, Compound motors with armature & field control |
| **Transformer** | Power transformer with flux animation, efficiency calculations |
| **PMSM** | Permanent Magnet Synchronous Motor with vector control visualization |

## 🚀 Getting Started

### Quick Start

1. **Visit the website:** [https://electromachinelab.com](https://electromachinelab.com)
2. **Choose a machine** from the homepage grid
3. **Adjust parameters** using sliders and controls
4. **Watch real-time** animations and waveforms
5. **Read theory** panels for deep explanations

### For Developers

#### Prerequisites
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Node.js (optional, for local development)

#### Local Development

```bash
# Clone the repository
git clone https://github.com/TanmoyAcharya/Mechatronics.git
cd Mechatronics

# Install dependencies (optional)
npm install

# Start local server
npx serve

# Visit http://localhost:3000
```

#### Deployment to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or connect your GitHub repository to Vercel for automatic deployments.

## 📖 How to Use

1. **Home Page** - Explore available machines or login/register
2. **Select Machine** - Click on any simulator card
3. **Adjust Parameters** - Use sliders to change voltage, current, speed, etc.
4. **Observe Results** - Watch real-time animations and waveforms
5. **Read Theory** - Expand theory panels for deep physics explanations
6. **Track Progress** - Create an account to save your learning progress

## 🔬 Physics Topics Covered

### Synchronous Machines
- Rotating magnetic fields
- Phasor diagrams & equivalent circuits
- Power-angle characteristics
- Synchronous reactance
- Voltage regulation
- Power factor correction

### Induction Machines
- Slip and asynchronous operation
- Torque-speed characteristics
- Equivalent circuit analysis
- Starting methods
- Speed control

### DC Machines
- Armature reaction
- Field control methods
- Speed regulation
- Back EMF visualization
- Motor characteristics

### Transformers
- Turn ratio & voltage transformation
- Magnetic flux visualization
- Equivalent circuits
- Efficiency calculations
- Loading effects

### PMSM
- Vector control (Id/Iq)
- d-q axis transformation
- Torque production
- Speed control

### Renewable Energy
- Wind turbine power curves
- Solar panel characteristics
- Maximum power point tracking
- V2G grid integration
